AI宣传广告项目书 - 视频添加指南

此文档说明如何将您的视频添加到项目书中，请按照以下步骤操作：

方法一：单个视频添加

如果您只需添加一个视频，可以使用以下代码：

// 打开浏览器控制台(F12)，粘贴以下代码：
addVideo('您的视频URL地址', '视频类型');

// 例如:
addVideo('https://example.com/your-video.mp4', 'video/mp4');

方法二：添加多个视频

如果您需要添加多个视频，可以使用以下代码：

// 打开浏览器控制台(F12)，粘贴以下代码：
addMultipleVideos([
    {
        title: '视频标题1',
        url: 'https://example.com/video1.mp4',
        type: 'video/mp4',
        description: '这是第一个视频的简短描述'
    },
    {
        title: '视频标题2',
        url: 'https://example.com/video2.mp4',
        type: 'video/mp4',
        description: '这是第二个视频的简短描述'
    },
    // 可以添加更多视频...
]);

微信视频格式支持

为确保在微信环境中正常播放，请注意：

  • 微信主要支持MP4格式视频
  • 视频大小应控制在20MB以内
  • 视频时长建议不超过5分钟
  • 视频分辨率建议使用720p及以下

视频托管建议

您可以使用以下服务托管您的视频：

  • 腾讯视频
  • 阿里云OSS
  • 七牛云
  • 又拍云

注意：完成视频添加后，建议您将此说明文件删除，以免影响项目书的专业性。
